How to add tickers by editing the code

Tip: The visual editor (Manage Tickers tab) is the easiest way to add tickers without touching code. Use this guide if you prefer to edit the HTML file directly — useful when adding many tickers at once.
① Locate the TICKERS object

Open index.html in any text editor. Search for const TICKERS = { — this is around line 140.

② Data structure for one ticker
"VTI": {
  name: "Vanguard Total Market",   // display name
  cat:  "Blend ETF",               // category (shown in dropdown group)
  data: {
    2015: { p: 0.009,  d: 0.019 }, // p = price return, d = dividend yield
    2016: { p: 0.125,  d: 0.020 },
    2017: { p: 0.213,  d: 0.018 },
    2018: { p: -0.053, d: 0.020 }, // negative = down year
    2019: { p: 0.307,  d: 0.019 },
    2020: { p: 0.212,  d: 0.016 },
    2021: { p: 0.255,  d: 0.013 },
    2022: { p: -0.193, d: 0.016 },
    2023: { p: 0.259,  d: 0.014 },
    2024: { p: 0.239,  d: 0.014 }
  }
},
③ Where to get the numbers
Price return (p) macrotrends.net → [TICKER] → "Annual Stock Return" — use Total Return column, divide by 100
Dividend yield (d) macrotrends.net → [TICKER] → "Dividend Yield" → Annual average — divide by 100. Use 0 for non-dividend stocks
Format Decimals, not percent: 12.5% → 0.125, -4.5% → -0.045, 3.2% → 0.032
④ Paste it in the right place

Add your new entry inside the TICKERS = { ... } object. Each entry ends with a comma except the last one:

const TICKERS = {
  "VYM": { ... },   // existing ticker
  "QQQ": { ... },   // existing ticker
  "VTI": { ... },   // <-- your new ticker (comma after it if more follow)
  "NVDA": { ... }   // last entry — no trailing comma
};
⑤ Save and refresh

Save the file, refresh the browser (or re-upload to GitHub Pages). Your ticker will appear automatically in both dropdowns, grouped by the cat field.

Common mistakes
p: 12.5 Wrong — this is 1250%, not 12.5%. Always use decimals.
Missing years If a year is missing, the overlap with other tickers shrinks. Fill in as many years as possible.
Trailing comma on last item Will cause a JS error. The last entry in TICKERS should NOT have a comma after the closing }.
d: undefined for growth If a stock pays no dividend, use d: 0, not omitting d entirely.
